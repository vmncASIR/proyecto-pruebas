# proyecto-pruebas

![Terraform Validate](https://github.com/vmncASIR/proyecto-pruebas/actions/workflows/terraform-validate.yml/badge.svg)

Proyecto de ejemplo para practicar versionado con Git tags y GitHub Releases, y validación automática con GitHub Actions.

## Versiones
- v1.0 - Versión inicial
- v1.1 - Estilos CSS y correcciones
- v1.2 - Página de contacto y navegación

## Infraestructura
- Terraform >= 1.0
- Provider AWS ~> 5.0
- LocalStack para simulación local de AWS
